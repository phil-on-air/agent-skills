# agent-skills

Public Hermes skills I share with friends.

## Included

- `creative/space-expanding-generation`

## Install

```bash
hermes skills tap add https://github.com/phil-on-air/agent-skills
```

## What this repo contains

A small curated subset of shareable skills, not my full private skill library.

Each skill lives in its own folder with a `SKILL.md` file.

## Highlight

### `creative/space-expanding-generation`

A method for generating genuinely different creative directions instead of collapsing too early to the safest polished default.

Useful for:

- fiction
- essays
- visual concepts
- art direction
- strategy
- research ideation

Core pattern:

1. generate structural spread
2. separate constraints from conventions
3. preserve anomalies
4. choose from a frontier

## Notes

These skills are designed for Hermes, but can also be read and used as standalone structured prompts.
